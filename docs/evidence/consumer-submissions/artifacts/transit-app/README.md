# Transit App Artifacts

Store only redacted, target-originated Transit App evidence here after a real
operator action occurs outside the repo.

Allowed examples: redacted receipt, ticket, email, portal screenshot, rejection
reason, blocker note, acceptance confirmation, or official-path verification
source note.

Do not add placeholder screenshots, fake receipts, fake tickets, fake emails,
example correspondence, portal credentials, private ticket links, tokens,
personal data, private operator artifacts, or screenshots with private portal
data.

After adding real evidence, update `../../current/transit-app.md`,
`../../README.md`, and `../../status.json` with only the evidence-backed status.
