# Real Agency GTFS Evidence

This directory contains templates only until real agency-approved, public-safe evidence exists.

Do not add placeholder GTFS ZIPs, fake validation outputs, fake approvals, fake import results, screenshots, private agency data, or private operator artifacts.

Use the templates here only after a real agency GTFS source has been reviewed for:

- source and permission
- license and contact metadata
- agency approval
- validation output redaction
- import and publish results
- public feed verification
- Phase 23 final-root status

If the raw source or raw validation output includes private paths, private contacts, private operator notes, non-public data, credentials, or private agency material, keep it private and commit only a redacted summary.

Current templates:

- `templates/import-review-template.md`: future real-agency GTFS import review packet.
